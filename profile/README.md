## ¿Quiénes somos?

**NIIOpay** es una plataforma tecnológica de pagos digitales con sede en Colombia, enfocada en construir la infraestructura financiera del futuro para Latinoamérica. Desarrollamos APIs robustas, pasarelas de pago seguras y aplicaciones que permiten a empresas de todos los tamaños procesar transacciones de forma rápida, segura y escalable.

## 🚀 Nuestros productos

| Producto | Descripción | Stack |
|---|---|---|
| **API NIIOpay** | API REST principal para procesamiento de pagos | TypeScript · Node.js |
| **App NIIOpay** | Aplicación móvil y web para gestión de pagos | TypeScript · React Native |
| **App CRM** | CRM interno para gestión de clientes | TypeScript · React |
| **Pay Gateway** | Pasarela de pagos multi-proveedor | Handlebars · Node.js |
