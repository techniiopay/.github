## ¿Quiénes somos?

**NIIOpay** es una plataforma tecnológica de pagos digitales con sede en Colombia, enfocada en construir la infraestructura financiera del futuro para Latinoamérica. Desarrollamos APIs robustas, pasarelas de pago seguras y aplicaciones que permiten a empresas de todos los tamaños procesar transacciones de forma rápida, segura y escalable.
